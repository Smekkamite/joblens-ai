# JobLens AI

JobLens AI is a Chrome extension that analyzes job postings and extracts structured insights using a local LLM via Ollama.

## What it does

JobLens reads job posts directly from LinkedIn and shows an AI-generated summary inside a custom side panel.

It extracts and displays:

- role
- work mode (remote / hybrid / onsite / unknown)
- experience
- salary
- tech stack
- verdict
- red flags

## Current Features

- LinkedIn job extraction
- Side panel UI injected into the page
- Ollama integration through a background service worker
- Structured JSON analysis generated by `llama3.1:8b`
- Local AI-powered analysis of job descriptions
